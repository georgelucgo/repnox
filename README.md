# 🏋️‍♂️ Repnox — Acompanhe sua Evolução

Aplicação web desenvolvida em React para registro e acompanhamento de treinos de musculação.
O objetivo é permitir que o usuário organize seus exercícios, registre cargas e repetições e acompanhe sua evolução ao longo do tempo.

---

## 🚀 Funcionalidades

* 📋 Registro de treinos (data, séries, peso e repetições)
* ✏️ Edição e exclusão de treinos
* ⭐ Sistema de favoritos
* 📊 Histórico completo de treinos
* 🔍 Visualização detalhada dos exercícios
* 💾 Persistência com LocalStorage
* 📱 Layout responsivo (mobile e desktop)

---

## 🖥️ Telas do Sistema

### 🏠 Home

* Apresentação da aplicação
* Sessão “Sobre”
* Navegação para os exercícios

### 💪 Exercícios

* Lista de exercícios organizados por grupo muscular
* Navegação para detalhes de cada exercício

### 📄 Detalhes

* Informações completas do exercício
* Imagens e tutorial
* Registro de treino
* Histórico específico do exercício

### 📊 Histórico

* Visualização geral dos treinos realizados
* Acompanhamento da evolução

### ⭐ Favoritos

* Lista de exercícios favoritados
* Acesso rápido aos treinos mais utilizados
